# IOCP-GameServer

게임 서버 프로그래밍 공부 내용을 정리하기 위한 리포지토리입니다.
실행을 위한 리소스는 포함되어 있지 않습니다.


# 프로젝트 설명
## 1. Game Client

WinAPI를 통해 제작한 기본적인 클라이언트 프로그램입니다. 아직 서버와의 연동은 되어 있지 않습니다.

- 체스 판을 그립니다.
- 체스 말 하나를 그립니다.
- 커서 키(WASD)로 말을 상하좌우로 이동합니다.
- 앞으로 작성할 서버 프로그램의 동작을 확인하기 위해 제작하였습니다.
